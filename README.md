<h1>프로젝트 이름</h1>
로또 번호 출력 및 맞추기
<h1>프로젝트 설명</h1>
랜덤으로 로또 번호를 생성하고 정렬하여 출력함
<h1>주제 선정 동기</h1>
로또 슬롯머신 프로젝트는 누구나 익숙한 로또를 소재로 해 별도의 설명 없이도 이해하기 쉽고, 
결과를 기다리는 과정에서 재미와 몰입감을 줄 수 있다고 판단해 선택했다. 
또한 간단한 아이디어로도 완성 시 눈에 보이는 결과와 성취감을 얻을 수 있는 주제라고 생각했다.
