# 🎰 로또 번호 출력 및 맞추기

간단한 콘솔 기반 로또 슬롯머신 프로젝트입니다.

사용자가 1~45 사이의 번호 6개를 입력하면 슬롯머신 연출 후 로또 당첨 번호를 생성하고, 자신의 번호와 비교하여 결과를 확인할 수 있습니다.

---

## 프로젝트 소개

본 프로젝트는 C 언어의 기초 문법을 활용하여 제작한 로또 시뮬레이션 프로그램입니다.

난수를 이용해 중복 없는 로또 번호를 생성하고, 생성된 번호를 정렬하여 출력합니다. 사용자가 입력한 번호와 비교하여 맞춘 개수와 등수를 계산하며, 슬롯머신 애니메이션을 통해 추첨 과정을 시각적으로 표현하였습니다.

---

## 주제 선정 동기

로또 슬롯머신 프로젝트는 누구나 익숙한 로또를 소재로 하여 별도의 설명 없이도 쉽게 이해할 수 있는 주제이다.

또한 결과를 기다리는 과정에서 재미와 몰입감을 줄 수 있으며, 비교적 간단한 아이디어만으로도 완성 후 눈에 보이는 결과와 성취감을 얻을 수 있다고 판단하여 선정하였다.

프로젝트를 진행하며 배열, 함수, 난수 생성, 정렬 알고리즘 등 프로그래밍의 기본 개념을 실제 프로그램에 적용해 볼 수 있다는 점도 주제 선정의 이유가 되었다.

## 주요 기능

### 로또 번호 생성

- 1~45 범위의 숫자를 랜덤으로 생성
- 중복 번호가 생성되지 않도록 처리
- 총 6개의 번호 추첨

### 번호 정렬

- 생성된 번호를 오름차순으로 정렬
- 사용자가 보기 쉽도록 출력

### 슬롯머신 연출

- 추첨 중 번호가 계속 바뀌는 효과 구현
- 실제 슬롯머신과 유사한 연출 제공

### 사용자 번호 입력

- 사용자가 직접 6개의 번호 입력
- 입력 범위 검증 가능

### 당첨 결과 확인

- 사용자 번호와 당첨 번호 비교
- 맞춘 개수 계산
- 결과에 따른 등수 출력

## 구현 과정

### 1. 사용자 번호 입력

사용자가 1~45 범위의 숫자 6개를 입력한다.

### 2. 로또 번호 생성

난수 함수를 이용하여 1~45 범위의 숫자를 생성한다.

이미 생성된 숫자와 중복될 경우 다시 생성하여 중복 없는 번호를 만든다.

### 3. 번호 정렬

생성된 번호를 오름차순으로 정렬하여 출력한다.

### 4. 슬롯머신 애니메이션

추첨 과정에서 번호가 계속 변경되는 것처럼 출력하여 슬롯머신 효과를 구현한다.

### 5. 당첨 결과 계산

사용자 번호와 당첨 번호를 비교하여 일치하는 번호 개수를 계산한다.

### 6. 등수 출력

맞춘 개수에 따라 최종 등수를 출력한다.
